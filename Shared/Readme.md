# Shared

Общий код, используемый несколькими микросервисами.

Используется строго ограниченно, чтобы избежать
распределённого монолита.

## Что здесь находится
- Абстракции (EventBus, Logging, Tracing)
- DTO и контракты для межсервисного взаимодействия
- Middleware и базовые инфраструктурные компоненты

## Чего здесь НЕ должно быть
- Domain модели
- Бизнес-логика
- Repository и DAL
- Зависимости на конкретные сервисы

## Принцип
Shared = *инфраструктурные building blocks*, а не бизнес
