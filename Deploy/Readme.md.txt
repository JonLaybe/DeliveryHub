# Deploy

Папка содержит всё, что относится к **развёртыванию системы в целевых окружениях**
(dev / stage / prod).

Здесь описывается **как задеплоить** сервисы и инфраструктуру,
но не содержится код самих сервисов.

## Что здесь находится
- Kubernetes manifests (k8s)
- Helm charts и values
- Ingress / Services / Secrets templates
- Конфигурации окружений

## Чего здесь НЕ должно быть
- Dockerfile сервисов
- docker-compose для локальной разработки
- бизнес-кода
- скриптов для локального запуска

## Принцип
Deploy = *операционная инфраструктура*
