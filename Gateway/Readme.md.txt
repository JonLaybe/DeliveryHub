# Gateway

API Gateway / BFF (Backend For Frontend) слой системы.

Gateway является единой точкой входа для внешних клиентов
(web, mobile, third-party).

## Ответственность
- Маршрутизация запросов
- Агрегация данных из нескольких сервисов
- Авторизация / аутентификация
- Rate limiting
- DTO для внешних клиентов

## Чего здесь НЕ должно быть
- Бизнес-логики домена
- Прямого доступа к БД сервисов
- Хранения состояния

## Принцип
Gateway = orchestration, не business logic
