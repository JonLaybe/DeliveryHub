# Docs

Документация проекта DeliveryHub.

Папка предназначена для хранения архитектурной,
технической и проектной документации.

## Что здесь находится
- Описание архитектуры системы
- Документация по микросервисам
- Контракты и события
- ADR (Architecture Decision Records)
- Диаграммы и схемы

## Принцип
Документация живёт рядом с кодом и обновляется вместе с ним.
