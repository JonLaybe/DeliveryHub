# UC-CL4: Просмотр товара

**Актор:** Покупатель (Гость или Авторизованный пользователь)

## Цель
Получить подробную информацию о товаре.

## Предусловия
- Каталог доступен (Catalog Service).
- Товар существует в системе.

## Основной сценарий
1. Пользователь выбирает товар в каталоге или по ссылке из поиска.
2. Запрашивает страницу товара.
3. Catalog Service загружает данные о товаре по ID.
4. Получает фото, характеристики, категорию и актуальное наличие (по запросу в Stock Service).
5. Возвращает детализированную информацию клиенту.

## Альтернативные сценарии
- **A1:** Товар не найден → 404 "Товар недоступен".
- **A2:** Ошибка Catalog Service → уведомление об ошибке.

## Постусловия
- Пользователь видит подробное описание товара.
