# UC-BS3: Очистить корзину

**Актор:** Покупатель (Авторизованный или Гость)

## Цель
Удалить все товары из корзины.

## Предусловия
- Корзина существует для пользователя.
- Basket Service доступен.

## Основной сценарий
1. Пользователь нажимает "Очистить корзину".
2. Basket Service удаляет все позиции из хранилища.
3. Корзина становится пустой.
4. Возвращается успешный ответ.

## Альтернативные сценарии
- **A1:** Корзина уже пуста → вернуть OK с пустым массивом.
- **A2:** Ошибка при обращении к хранилищу → вернуть ошибку.

## Постусловия
- Корзина пользователя очищена.
