# UC-BS2: Просмотреть корзину

**Актор:** Покупатель (Авторизованный или Гость)

## Цель
Просмотреть текущие товары, добавленные в корзину.

## Предусловия
- Коллекция/хранилище корзины доступно (Basket Service или кеш).

## Основной сценарий
1. Пользователь открывает "Корзину".
2. Frontend отправляет запрос к Basket Service.
3. Basket Service загружает данные корзины по userId/cookieId.
4. Возвращает товары, их количество и актуальные цены (по необходимости запрашивая Catalog).
5. Клиент отображает содержимое корзины.

## Альтернативные сценарии
- **A1:** Корзина пуста → сообщение "Корзина пуста".
- **A2:** Ошибка соединения → предложение повторить позже.

## Постусловия
- Пользователь получил актуальное состояние корзины.
