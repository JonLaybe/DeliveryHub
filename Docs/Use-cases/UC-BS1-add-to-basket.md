# UC-CL3: Добавление товара в корзину

**Актор:** Покупатель  
**Цель:** Добавить выбранный товар в корзину.

## Предусловия
- Товар существует в каталоге.
- Имеется доступ к сервису корзины (Basket Service).
- Есть информация о наличии товара в Stock Service.

## Основной сценарий
1. Пользователь нажимает "Добавить в корзину".
2. UI отправляет запрос на API Gateway.
3. Gateway перенаправляет запрос в Basket Service.
4. Basket Service:
   - проверяет существование товара (Catalog Service)
   - запрашивает остатки в Stock Service
   - резервирует товар при наличии
5. Корзина обновляется, пользователю возвращается новое состояние.

## Альтернативные потоки
- **4а. Нет в наличии** → сообщение "Товара нет на складе".
- **3а. Сервис корзины недоступен** → ошибка с предложением повторить позже.

## Постусловия
- Товар зарезервирован, добавлен в корзину.
