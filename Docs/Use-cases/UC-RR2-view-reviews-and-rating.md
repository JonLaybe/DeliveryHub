# UC-RR2 — Просмотреть отзывы и рейтинг

**Actor:** Любой пользователь (гость или авторизованный)  
**Goal:** Познакомиться с мнением других покупателей о товаре  
**Primary System:** Review & Rating Service

---

## Preconditions
- Товар существует

---

## Main Flow
1. Пользователь открывает карточку товара
2. Фронтенд запрашивает отзывы и среднюю оценку
3. Система возвращает список отзывов и рейтинг
4. Интерфейс отображает данные пользователю

---

## Alternative Flows
**3A.** Отзывов нет → возвращается пустой список, UI отображает “Нет отзывов”

---

## Postconditions
- Нет (read-only)
