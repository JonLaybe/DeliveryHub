# Scripts

Вспомогательные скрипты для разработчиков и CI/CD.

Папка содержит утилиты, которые автоматизируют
частые или одноразовые операции.

## Что здесь находится
- Скрипты запуска и остановки системы локально
- Reset окружения (volumes, контейнеры)
- Наполнение БД тестовыми данными
- Вспомогательные Dev / CI операции

## Примеры
- start-local.sh / .ps1
- stop-local.sh
- seed-dev-data.sh
- reindex-search.sh

## Чего здесь НЕ должно быть
- Бизнес-логики
- Production-deploy сценариев
- Dockerfile и manifests

## Принцип
Scripts = *что сделать*, а не *как устроено*
